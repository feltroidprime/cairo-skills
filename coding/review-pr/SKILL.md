# Review PR Skill

This skill provides comprehensive pull request reviews with actionable feedback.

## Features

- Code quality analysis
- Security vulnerability detection
- Performance suggestions
- Style and convention checks
- Inline comments with suggestions

## Usage

Invoke this skill when you want to review a pull request. It will:
1. Fetch the PR diff
2. Analyze changes for issues
3. Provide structured feedback
4. Suggest improvements
