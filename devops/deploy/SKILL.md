# Deploy Skill

This skill assists with deployment processes and helps manage CI/CD pipelines.

## Capabilities

- Generate deployment configurations
- Create CI/CD pipeline definitions
- Validate deployment manifests
- Environment variable management
- Rollback assistance

## Supported Platforms

- GitHub Actions
- GitLab CI
- Docker/Kubernetes
- Cloud platforms (AWS, GCP, Azure)
